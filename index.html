Политика конфиденциальности

Дата последнего обновления: 16.01.2026

Приложение «My Dictionary» уважает конфиденциальность пользователей и стремится обеспечивать защиту персональных данных.

1. Какие данные собираются

1.1 Локальные данные

  • словари и списки слов
  • переводы слов и фраз
  • карточки для повторения
  • частота использования слов

1.2 Данные при входе через Google

  • имя пользователя
  • адрес электронной почты
  • идентификатор Google-аккаунта

1.3 Данные при синхронизации

  • словари пользователя
  • слова, фразы и переводы
  • настройки приложения
  • технические данные (IP, токены, user-agent)

2. Использование данных

Данные используются исключительно для работы приложения и синхронизации.

3. Защита данных

Мы применяем разумные технические меры для защиты данных пользователя.

4. Удаление данных

Пользователь может удалить свои данные через настройки приложения.

Privacy Policy

Last updated: 16.01.2026

My Dictionary respects user privacy and is committed to protecting personal data.

1. Data we collect

1.1 Local data

  • dictionaries and word lists
  • translations
  • flashcards
  • word usage statistics

1.2 Google Sign-In data

  • user name
  • email address
  • Google account ID

1.3 Sync data

  • user dictionaries
  • words and translations
  • app settings
  • technical auth data (IP, tokens, user-agent)

2. Data usage

Data is used only for app functionality and synchronization.

3. Data protection

We apply reasonable technical measures to protect user data.

4. Data removal

User can delete data via app settings.
