Политика конфиденциальности

Дата последнего обновления: 16.01.2026

Приложение «My Dictionary» уважает конфиденциальность пользователей и стремится обеспечивать защиту персональных данных.

1. Какие данные собираются

1.1 Локальные данные

  • словари и списки слов
  • переводы слов и фраз
  • карточки для повторения
  • частота использования слов

1.2 Данные при входе через Google

  • имя пользователя
  • адрес электронной почты
  • идентификатор Google-аккаунта

1.3 Данные при синхронизации

  • словари пользователя
  • слова, фразы и переводы
  • настройки приложения
  • технические данные (токены, IP, user-agent)

2. Использование данных

Данные используются исключительно для работы приложения, синхронизации и восстановления данных.

3. Защита данных

Применяются разумные технические меры для защиты данных.

4. Удаление данных

Пользователь может удалить свой аккаунт в настройках приложения.

После запроса на удаление аккаунта:

  • доступ к аккаунту и данным немедленно блокируется
  • аккаунт помечается как удалённый
  • данные становятся недоступны для использования и синхронизации

Окончательное удаление данных с серверов производится в течение 30 дней с момента запроса на удаление аккаунта.

После удаления аккаунта и по истечении 30 дней персональные данные пользователя удаляются полностью и без возможности восстановления.

При этом приложение может сохранять обезличенные данные, которые не позволяют идентифицировать пользователя и не могут быть связаны с конкретной личностью.

К таким данным могут относиться:

  • обобщённые словари и списки слов
  • переводы слов и фраз без привязки к аккаунту
  • статистические и аналитические данные использования

Обезличенные данные используются исключительно для анализа, улучшения работы приложения и развития функциональности.

Privacy Policy

Last updated: 16.01.2026

The "My Dictionary" app respects user privacy and is committed to protecting personal data.

1. What data is collected

1.1 Local data

  • dictionaries and word lists
  • word and phrase translations
  • flashcards
  • word usage statistics

1.2 Google Sign-In data

  • user name
  • email address
  • Google account ID

1.3 Sync data

  • user dictionaries
  • words, phrases and translations
  • app settings
  • technical data (tokens, IP, user-agent)

2. Data usage

Data is used only for app functionality, synchronization and recovery.

3. Data protection

We apply reasonable technical measures to protect user data.

4. Data deletion

The user may delete their account via the app settings.

After requesting account deletion:

  • access to the account and data is immediately blocked
  • the account is marked as deleted
  • all data becomes unavailable for use and synchronization

Final deletion of user data from our servers is performed within 30 days after the deletion request.

After account deletion and the expiration of the 30-day period, all personal user data is permanently deleted and cannot be restored.

At the same time, the application may retain anonymized data that does not allow identification of the user and cannot be linked to a specific individual.

Such anonymized data may include:

  • aggregated dictionaries and word lists
  • word and phrase translations without account association
  • statistical and analytical usage data

Anonymized data is used solely for analytics, service improvement and enhancement of application functionality.
