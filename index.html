Политика конфиденциальности приложения «My Dictionary»

Дата последнего обновления: 16.01.2026

Приложение «My Dictionary» уважает конфиденциальность пользователей и стремится обеспечивать защиту персональных данных. Настоящая Политика конфиденциальности описывает, какие данные собираются, как они используются и каким образом защищаются.

1. Какие данные собираются

1.1 Локальные данные

По умолчанию все данные пользователя хранятся локально на устройстве:

  • словари и списки слов
  • переводы слов и фраз
  • карточки для повторения
  • частота использования слов

Эти данные не передаются на сервер без авторизации пользователя.

1.2 Данные при входе через Google

При авторизации через Google может быть получена следующая информация:

  • имя пользователя
  • адрес электронной почты
  • идентификатор Google-аккаунта

1.3 Данные, передаваемые на сервер при синхронизации

  • словари пользователя
  • слова, фразы и переводы
  • настройки приложения
  • технические данные для авторизации (токены, устройство, IP, user-agent)

2. Как используются данные

Данные используются исключительно для работы функционала приложения, синхронизации и восстановления данных.

3. Хранение и защита данных

Мы применяем разумные технические меры для защиты данных пользователя.

4. Удаление данных

Пользователь может удалить свои данные через настройки приложения или запросить удаление аккаунта.
